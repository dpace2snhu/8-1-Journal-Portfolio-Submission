The main problem I was solving in this course was creating a program that would help academic advisors quickly access course information for students. 
The program needed to read course data from a file, store it in a data structure, print a sorted list of courses, and allow advisors to look up individual courses and their prerequisites. 
The goal was to make it easier to find information without having to search through a file manually.
To solve this problem, I first looked at the different data structures we learned about and compared their strengths and weaknesses. 
I chose a binary search tree because it allows courses to be stored in order and makes searching for a specific course more efficient. 
This project showed me why understanding data structures is important. 
The data structure you choose can affect how quickly information can be found and how easy it is to organize and manage data.
One of the biggest challenges I faced was getting comfortable with implementing data structures in C++ and making sure the program correctly read and validated the course file. 
I also ran into issues while testing and debugging the program. 
When I encountered problems, I worked through them one step at a time, reviewed my code carefully, and used the feedback from previous assignments to make improvements. 
Breaking the project into smaller pieces made it much easier to manage.
This project helped me think more about planning before writing code. 
Instead of immediately jumping into programming, I learned the value of creating pseudocode, comparing possible solutions, and thinking about how the program would work as a whole. 
Taking the time to design the program first made the coding process much smoother and helped me better understand how different parts of the program work together.
My work in this course has also changed the way I write code. 
I now pay more attention to using meaningful variable names, creating separate functions for specific tasks, and adding comments where they are helpful. 
These practices make programs easier to read, maintain, and update in the future. 
Overall, this project strengthened my understanding of data structures and gave me more confidence in designing and building programs from start to finish.
